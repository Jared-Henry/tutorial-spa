# Angular & .Net Core SPA
Create an Angular and .Net Core SPA in under thirty minutes:
* Angular CLI
* Lazy loaded TypeScript modules
* CORS
* MVC Web API
* Entity w/ Migrations
* Automated Build, Testing, & Release
